typedef hash_table_t commit_t

A commit is a hash table. The key is the name of the key, i.e. the hash of the work tree is associated with the key "COMMIT_KEY_HASH" (that is defined in commit.h).

Parameters
key_valThe array of key_val_t pointers
max_sizeThe maximum size of the array (i.e. the number of key_val_t pointers that can be stored in the array)
sizeThe current size of the array (i.e. the number of key_val_t pointers that are currently stored in the array, and that are not NULL)
