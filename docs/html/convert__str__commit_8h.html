char * convert_commit_to_str ( commit_t *  commit)

Convert the commit provided to a string. The string is allocated, and must be freed by the caller. The string is formatted as follow: "KEY1:VALUE1\nKEY2=VALUE2\nKEY3=VALUE3\n" etc. The format between newlines is defined in the convert_key_val_to_str function.

Parameters
commitThe commit to convert to a string.
Returns
char* The string representing the commit. NULL if an error occurred.
